Technology

Teknologi semakin canggih terutama internet maka dari itu diciptakannya alat yang dapat membantu kita dalam melakukan sesuatu yang kita butuhkan. Dalam pembuatan Sistem Monitoring dan Control ini menggunakan banyak tools, baik itu dari segi software dan hardware.

Hardware : Arduino Uno R3, TTL Serial Camera VC0706, DHT11, MQ-2, Relay, Kabel Jumper, Project Board,
  Kabel LAN (UTP) Tipe Straigh, Ethernet Shield W5100, Kabel USB Tipe A/B, Resistor 10k Ohm.

Network : Modem atau sumber internet lainnya yang di hubungkan ke PC/Laptop sebagai WebServer.

Software - Aplikasi : Xampp, Web Browser, Sublime Text 3.
         - Bahasa Pemgrograman : C, PHP, SQL, Javascript.
         - Bahasa Markup : HTML, CSS.
         - Framework : Bootstrap, JQuery, DataTables, HighCharts, Particles JS, Typed JS, Blueimp Gallery.
         - Icon : icon8, iconfinder, Font Awesome, Bootstrap Icon.
Network : Virtual Private Server (VPS) untuk membangun koneksi Virtual Private Network (VPN) sebagai alternatif menyewa IP Public.
